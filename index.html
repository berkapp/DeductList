Admin code unlocks full edit access. A division code opens a read-only view of that division's rows only.

Data autosaves to this browser (localStorage) as you type or add/delete rows. Monthly JAN–DEC figures start blank — enter them yourself as deductions actually happen. Pending Deduction = amount owed at creation. Total Deducted = sum of JAN–DEC entered so far. Balance Deduction = Pending Deduction − Total Deducted.
